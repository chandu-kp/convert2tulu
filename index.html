Instructions

  • Writing In Kannada

    Easiest way is to download and install Google Input Tool chrome extension or if using mobile, use Gboard - the Google Keyboard. To display both in Kannada and Tulu script, use # like below:

    ನಮಸ್ಕಾರ #ನಮಸ್ಕಾರ#

  • Customizing Text

    The text can be customized by pressing the settings (settings) icon. You can change the font color, font size, outline color, weight. Also, you can change background color or upload your own background image.

  • Downloading Image

    Once you have finished writing, click on Download button to download the content as image.
